# Contributing Guidelines

This project is developed by Adiyan Hossain and Andrew Tan for CECS 478.

## Workflow
- Use feature branches for development.  
- Write clear commit messages.  
- Open pull requests for partner review.  
- Do not commit keys, secrets, or sensitive data.  

## Roles
- Andrew Tan: Networking logic, server/client behavior.  
- Adiyan Hossain: Encryption layer, Docker setup, documentation.
